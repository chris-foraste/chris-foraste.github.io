This project utilizes 2 PCBs I previously designed and soldered - a voltage regulator to provide clean power to the KB 2040 and an H-bridge to drive the stepper motors. Both PCBs were augmented to enable barrel jack inputs and LEDs to visualize current flow.
Numerous construction techniques were used to manufacture the maze, including a chop saw, SOLIDWORKS, 3D Printing, laser cutter, robotic tap, jump shear, rivet gun, drill press, v-bender, and power drill.

Code

This code was designed with usability in mind as it sought to balance the user's desire for ease of control with the mechanical desire to constrain the rotation of the pulleys as to not ram the maze into the ceiling or the floor (this was a frequent issue during usability testing).

After all the pins are assigned, the code consists of a state machine which allows the spools to operate independently of each other, allowing for more natural control and greater freedom in terms of movement.

This rotation is then tracked internally, and if the user attempts to turn the spool further, the motor simply will not receive an input, causing the user to reverse course. I predicted that this system might not work consistently all of the time and so I added in a failsafe button which can be pressed to allow three seconds to reset the spools to a neutral position before tracking restarts.
