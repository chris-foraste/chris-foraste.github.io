Pinball Table

A fully functional pinball table created using Woodworking, CAD, and Arduino skills

Project Description:

This pinball machine was the culmination of the work I put into my honors engineering class in high school. The game board is built from a combination of plywood and pine using drills, a table saw, a miter saw, and a band saw. The game paddles were 3D printed using a custom design I created, the technical drawing of which is shown in the right-hand column. Lastly, the sensing mechanisms was built using Arduino components. We drilled small holes in the game board and placed photoresistors in them in order to detect the ball's location. As the ball passed over, it created a shadow, triggering a change in the photoresistor and alerting our system to the location.
